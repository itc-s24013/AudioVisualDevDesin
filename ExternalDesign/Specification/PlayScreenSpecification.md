# 画面仕様書

## 画面ID: SCR-03
## 画面名: 再生画面

## 目的
アップロードした音声ファイルを、選択したプリセットのデザインでオーディオビジュアライザーを再生する。

## レイアウト

```
+-------------------------------------------------------------+
| [戻る]                                                      |
| +---------------------------------------------------------+ |
| |                                                         | |
| |                                                         | |
| |            選択したプリセットのデザインで再生           | |
| |                                                         | |
| |                                                         | |
| |---------------------------------------------------------+ |
| | 再生バー [10秒戻す] ((再生・停止ボタン)) [10秒とばす]   | |
| +---------------------------------------------------------+ |
+-------------------------------------------------------------+
```

上部:戻るボタン
中央:選択したプリセットのデザインによるオーディオビジュアライザー表示エリア
下部:再生バー、10秒戻す、再生・停止ボタン、10秒とばす

## 入力項目

| 項目名 | 型 | 必須 | バリデーション |
|---|---|---|---|
| 再生バー(シーク位置) | 数値(再生時間) | 任意 | 音声ファイルの総再生時間の範囲内であること/再生が終了したら0:00の位置に戻し一時停止する |

## ボタン・リンク

| ボタン名 | 動作 | 遷移先画面ID |
|---|---|---|
| 戻る | 再生を停止し、直前の画面に戻る | SCR-02 |
| 再生・停止ボタン | 音声の再生/一時停止を切り替える/再生が終了したら一時停止 | 遷移なし(同画面内表示更新) |
| 10秒戻す | 再生位置を10秒前に戻す | 遷移なし(同画面内表示更新) |
| 10秒とばす | 再生位置を10秒先に進める | 遷移なし(同画面内表示更新) |
| 再生バー(ドラッグ操作) | 任意の再生位置にシークする | 遷移なし(同画面内表示更新) |

## 表示項目

| 項目名 | データソース(情報モデル) |
|---|---|
| オーディオビジュアライザー | 選択中プリセットの詳細情報モデル、および再生中音声ファイルの音声解析データ |
| 再生バー(現在位置/総再生時間) | 再生中音声ファイルのメタデータ・再生状態 |
| 再生・停止状態 | 再生状態モデル(再生中/停止中) |

## エラー時挙動

- **入力エラー**: 再生バーのシーク位置が音声ファイルの範囲外に指定された場合、範囲内に補正して処理を継続する。
- **通信エラー**: 音声ファイルの読み込みやビジュアライザー用データの取得に失敗した場合、エラーメッセージを表示し、SCR-02 ホーム画面への復帰を促す。
- **セッション切れ**: セッションが無効な状態で操作が行われた場合、SCR-01 ログイン画面へ遷移し、再ログインを促す。
