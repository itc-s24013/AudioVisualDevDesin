# 画面仕様書

## 画面ID: SCR-02
## 画面名: ホーム画面

## 目的
ユーザが音声ファイルをアップロードし、プリセットを選択・保存・管理して、デザイン設定を行えるようにする。

## レイアウト

```
+------------------------------------------------------------+----------------------------+
|                                                            |         ユーザ名           |
|                 +-----------------------+                  |       プリセット名         |
|                 |  ファイルアップロード |                  |  +----------------------+  |
|                 +-----------------------+                  |  |                      |  |
|                                                            |  |                      |  |
|  +-----------+----+  +-------+  +-------+  +-------+       |  |  デザイン設計パネル  |  |
|  | 保存済みの|... |  |       |  |       |  |       |       |  |                      |  |
|  | プリセット+----+  |       |  |       |  |       |       |  |                      |  |
|  +----------------+  +-------+  +-------+  +-------+       |  |                      |  |
|                                                            |  +----------------------+  |
|  +----------------+  +-------+  +-------+  +-------+       |  +--------------+ +------+ |
|  |                |  |       |  |       |  |       |       |  |シミュレーター| | 保存 | |
|  +----------------+  +-------+  +-------+  +-------+       |  +--------------+ +------+ |
+------------------------------------------------------------+----------------------------+
```

左側:ファイルアップロード領域とプリセット一覧(グリッド表示)
右側:ユーザ名表示、プリセット名編集、デザイン設計パネル、シミュレーター、保存ボタン

## 入力項目

| 項目名 | 型 | 必須 | バリデーション |
|---|---|---|---|
| アップロードファイル | ファイル(音声) | 任意 | 対応音声フォーマットのみ許可、ファイルサイズ上限あり |
| プリセット名 | 文字列 | 必須 | 未入力の場合は空白表示で保存不可。保存済みプリセットの名前変更にも使用。 |
| デザイン設計パネル入力値 | 文字列/数値(パネル内項目による) | 任意 | パネル内の各項目に応じたバリデーション |

## ボタン・リンク

| ボタン名 | 動作 | 遷移先画面ID |
|---|---|---|
| ファイルアップロード | ファイル選択ダイアログを開き、音声ファイルをアップロードする | SCR-03(アップロード完了後) |
| 保存済みのプリセット(各カード) | 選択したプリセットの情報をデザイン設計パネルに表示する。(再選択で選択解除) | 遷移なし(同画面内表示更新) |
| シミュレーター | 現在のデザイン設定内容を視聴・確認する | 遷移なし(同画面内、モーダル等で表示) |
| 保存 | 現在のプリセット名・デザイン設定内容を保存する(未選択ならプリセット新規作成) | 遷移なし(同画面内、保存完了通知) |
| 削除 | 「...」を選択すると出現。プリセットのdeletedフラグをtrueに変更 | 遷移なし(同画面内、削除完了通知、削除済みプリセットを非表示) |
| プリセット名の編集 | 「...」を選択すると出現。プリセット名を変更 | 遷移なし(同画面内、更新通知、プリセット名を変更) |

## 表示項目

| 項目名 | データソース(情報モデル) |
|---|---|
| ユーザ名 | ユーザ情報モデル(ログイン中ユーザ) |
| プリセット名 | プリセット情報モデル |
| 保存済みのプリセット一覧 | プリセット情報モデル(ユーザに紐づく一覧) |
| デザイン設計パネルの内容 | 選択中プリセットの詳細情報モデル |

## エラー時挙動

- **入力エラー**: プリセット名やデザイン設計パネルの入力値が不正な場合、該当項目にエラーメッセージを表示し、保存を中断する。
- **通信エラー**: ファイルアップロードやプリセット保存時にサーバとの通信に失敗した場合、エラーメッセージを表示し、再試行を促す。
- **セッション切れ**: セッションが無効な状態で操作が行われた場合、SCR-01 ログイン画面へ遷移し、再ログインを促す。
